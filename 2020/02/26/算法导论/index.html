gale-shaply algorithm

初始化：

while （存在h没有匹配 && 没对每一个s发出申请）

{

s=h还没申请的学生列表中的第一个

if（s没配对）

h-s配对

else if（s prefer h to current h’)

h’-s 替换为h-s

else

s拒绝h

}

return M

gale-shaply algorithm：

至多n^2^次迭代while

至多n2次申请：n（n-1）+1

证明gsa所得的M种没有unstable pair：

考虑任意配对h-s ，不在M中：

1 h没有向s发出申请过：

因为h的申请是从好到坏的，所以h比起s，更偏好它目前的s’

2 h向s发出过申请：

s 拒绝过h

s 比起h，更喜欢它目前的h’

所以h-s不是unstable的
